#  Heart Disease Classification

A machine learning project that predicts the presence of heart disease in patients based on clinical data.

## Overview

This project applies supervised classification algorithms to the UCI Heart Disease dataset to determine whether a patient is at risk of heart disease. It covers the full ML pipeline: data exploration, preprocessing, model training, evaluation, and visualization.

## Features

- Exploratory Data Analysis (EDA) with visualizations
- Data preprocessing and feature engineering
- Multiple classification models (e.g. Logistic Regression, Random Forest, SVM, XGBoost)
- Model evaluation using accuracy, precision, recall, F1-score, and ROC-AUC
- Hyperparameter tuning for optimized performance

## Dataset

Uses the [UCI Heart Disease Dataset](https://archive.ics.uci.edu/ml/datasets/Heart+Disease) (Cleveland subset), containing 13 clinical features such as age, cholesterol, chest pain type, and resting blood pressure.

## Results

Models are compared based on key classification metrics to identify the best-performing approach for heart disease prediction.
